# MoneyMatch3

f2p hybrid-casual **match-3** для РуСтор (вариант «соединение цепочки»). Игрок ведёт пальцем по
цепочке соседних одинаковых денежных плиток (≥2) — они собираются в **Баланс**; сверху
досыпаются новые (гравитация). Сверху сидит Уоррен Баффет «на деньгах» и реагирует на комбо.
Позже из Баланса можно будет покупать инвестиции, повышающие ценность собираемых денег.

> Самостоятельный проект на каркасе `08_MergeMoney` (Vite/TS/DOM, 384×844), но с другой
> кор-механикой и сейвом. Это два независимых проекта.

## Запуск

```bash
npm install
npm run dev        # http://localhost:8778
npm run build      # tsc --noEmit && vite build → dist/
npm run typecheck
```

## Стек

TypeScript + Vite, чистый DOM/CSS (без игрового фреймворка). UI — абсолютно позиционированные
DOM-вью внутри `#stage` 384×844 (FIT-масштаб). Анимации — Web Animations API. Логика —
pure-функции `core/*`.

## Управление

- **Сбор:** зажми плитку и проведи по цепочке соседних одинаковых (≥2) — отпусти, чтобы собрать.
- **Откат:** веди палец назад на предпоследнюю плитку.
- **DEV-панель** (только в dev-сборке): кнопка «DEV» или хоткей `~`.
