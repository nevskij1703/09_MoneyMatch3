# 08 — Отложенные фичи (roadmap)

На текущей итерации это ЗАГЛУШКИ или ещё не начато. Сгруппировано по областям.

## Уоррен Баффет (верхняя зона)

- Сейчас: статичная фигура «сидит на деньгах» + декоративные падающие деньги +
  лёгкий `popReaction` (эмодзи-пузырь + подскок) на комбо ≥ 4 ([buffettView.ts](../../src/ui/dom/buffettView.ts)).
- Дальше: **анимированный** Баффет, набор реакций на разные комбо/события (рандомные реплики,
  эмоции, спец-анимации на крупные цепочки), арт вместо эмодзи.

## Инвестиции (трата Баланса)

- Сейчас: вкладка «Инвестиции» в тулбаре → `stubModal`.
- Дальше: игрок тратит **Баланс** на инвестиции, которые поднимают `investmentMultiplier`
  (ценность всех будущих сборов). Возможны ветки/уровни вложений, пассивные эффекты.
  `SaveData.investmentMultiplier` уже заведён под это.

## Бустеры на поле (расширение)

- Сейчас: 💣 **Бомба** (из фигуры T/L → взрыв 3×3), 🚀 **Ракета** (из квадрата 2×2 → ряд/столбец,
  h/v рандом) и 🧲 **Магнит** (из линии-5 → весь тир) — самостоятельные объекты, с цепной
  реакцией; свайп двух бустеров активирует оба. См. [01_core_loop.md](01_core_loop.md).
- Дальше:
  - **Усиленное комбо двух бустеров** при свапе друг с другом (Бомба+Бомба = взрыв 5×5,
    Ракета+Ракета = крест, Магнит + бустер = супер-сбор) — сейчас оба срабатывают независимо,
    без объединённого эффекта.
  - **Бустер за линию из 4** (напр. отдельная «Молния») — намеренно НЕ сделан (линия-4 =
    обычный матч).
  - VFX взрыва / полёта ракеты / притяжения магнита (сейчас — общий pop + collect-FX).

## Инвентарные бустеры (4 кнопки внизу)

- Сейчас: 4 кнопки-заглушки (shuffle / hammer / lightning / magnet), счётчик инвентаря, тап → `stubModal`.
  Инвентарь хранится в `SaveData.boosters`; `shuffleBoard` уже реализован (используется анти-дедлоком).
- Дальше: реальные эффекты «помочь собрать деньги» — перемешать поле, разбить плитку,
  собрать линию/ряд, собрать все плитки одного тира; цены в 💎 / Балансе; анимации.

## Прогрессия (новые элементы / тиры)

- Сейчас: фиксированный набор T1..4 (`tierCount`).
- Дальше: с ростом (Баланс / `totalCollected`) разблокировать новые тиры/типы плиток,
  усложнять поле. Хук: `tierCount` + `totalCollected` в сейве.

## Мета и сервисы (фаза релиза)

- Магазин (💎, бустер-паки), Портфель, Задачи (дейлики) — сейчас вкладки-заглушки.
- SDK-стабы, Store_Info, keystore, build-config, privacy policy, RC-сборка — при подготовке к
  публикации в РуСтор (канон — в глобальном `~/.claude/`). Во время разработки не трогаем.
