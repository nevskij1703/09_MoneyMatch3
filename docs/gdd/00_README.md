# MoneyMatch3 — Game Design Document

f2p hybrid-casual **классический match-3** для РуСтор. Кор-механика — **свайп-обмен** двух
соседних денежных плиток: линии из 3+ по верт./гориз. и квадраты 2×2 авто-схлопываются в
Баланс, спецматчи (2×2, линия-5) рождают спецтайлы-бустеры на поле; сверху досыпаются новые
плитки (гравитация, каскады). Сверху сидит Уоррен Баффет «на деньгах» и реагирует на комбо.

## Глоссарий

- **Плитка / тир** — денежный объект на поле 5×5 (старт T1..T4; внутренняя стоимость `2^t`).
- **Свап** — обмен двух ортогонально соседних плиток свайпом; валиден, если создаёт матч
  (линию ≥ `minLine` или квадрат 2×2) или затрагивает спецтайл, иначе откатывается.
- **Матч** — линия ≥ `minLine` (=3) по верт./гориз. ИЛИ квадрат 2×2 одного тира.
- **Спецтайл** — бустер на поле: 💣 **Бомба** (из 2×2 → взрыв 3×3), 🧲 **Магнит** (из линии-5 →
  сносит весь тир). Срабатывает при свайпе с соседом или попадании под схлоп (цепная реакция).
- **Каскад** — повторный схлоп после досыпки в рамках одного хода.
- **Комбо** — счётчик натуральных схлопов за ход (взрывы спецтайлов НЕ считаются): «Комбо», «Комбо
  ×2», … — крупный баннер над полем + растущий денежный бонус (+5%, +6%, +7%, …).
- **Баланс** — главная валюта. Растёт на ценность схлопнутых плиток.
- **💎 (алмазы)** — премиум-валюта (на будущие цены бустеров).
- **Гравитация** — после схлопа уцелевшие плитки падают вниз, сверху досыпаются новые.
- **investmentMultiplier** — множитель ценности сбора (будущие инвестиции его поднимают; старт 1).

## Текущая итерация

Реализована рабочая кор-петля классического match-3: свайп-обмен, схлоп линий 3+ и квадратов
2×2, спецтайлы (Бомба 3×3, Магнит — весь тир) с цепной реакцией, каскады с растущим комбо,
анти-дедлок, чистый старт. Заглушки: Баффет (статичная фигура + лёгкая реакция), 4 инвентарных
бустера, инвестиции, вкладки тулбара. Прогрессия тиров (новые элементы) — будущее.
