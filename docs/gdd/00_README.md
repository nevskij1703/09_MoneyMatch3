# MoneyMatch3 — Game Design Document

f2p hybrid-casual **match-3** для РуСтор. Кор-механика — «соединение цепочки»: игрок ведёт
пальцем по цепочке соседних одинаковых денежных плиток и собирает их в Баланс; сверху
досыпаются новые (гравитация). Сверху сидит Уоррен Баффет «на деньгах» и реагирует на комбо.

## Разделы

- [01_core_loop.md](01_core_loop.md) — кор-петля: цепочка → сбор → Баланс → гравитация/досыпка.
- [08_future.md](08_future.md) — отложенные фичи: анимированный Баффет и реакции, инвестиции
  (трата Баланса), 4 бустера, прогрессия тиров, мета-системы.

## Глоссарий

- **Плитка / тир** — денежный объект на поле (старт T1..T4; внутренняя стоимость `2^t`).
- **Цепочка** — последовательность смежных (ортогонально) плиток ОДНОГО тира, набираемая
  drag-ом; собирается при длине ≥ `minChain` (=2).
- **Баланс** — главная валюта. Растёт на ценность собранной цепочки.
- **💎 (алмазы)** — премиум-валюта (на будущие цены бустеров).
- **Гравитация** — после сбора уцелевшие плитки падают вниз, сверху досыпаются новые тех же тиров.
- **investmentMultiplier** — множитель ценности сбора (будущие инвестиции его поднимают; старт 1).
