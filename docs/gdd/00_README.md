# MoneyMatch3 / «Hamster Bank» — Game Design Document

f2p hybrid-casual **классический match-3** для РуСтор. Бренд экрана — **«Hamster Bank»**
(маскот-хомяк-банкир, синяя тема). Кор-механика — **свайп-обмен** двух соседних денежных
плиток: линии из 3+ по верт./гориз. и квадраты 2×2 авто-схлопываются в **Баланс**; сверху
досыпаются новые плитки (гравитация, каскады, комбо). Бустеры — **кнопки-инвентарь** под полем
(на поле НЕ спавнятся). Над полем — карта баланса с маскотом, офферы и строка Level/Energy/Income.

> Кодовая база и сейв остаются `MoneyMatch3` / `mmatch_save`. «Hamster Bank» — отображаемый
> бренд главного экрана (макет Figma «Play window»).

## Глоссарий

- **Плитка / тир** — денежный объект на поле **6×5** (старт T1..T4; внутренняя стоимость `2^t`).
- **Свап** — обмен двух ортогонально соседних плиток свайпом; валиден, если создаёт матч
  (линию ≥ `minLine` или квадрат 2×2), иначе откатывается.
- **Матч** — линия ≥ `minLine` (=3) по верт./гориз. ИЛИ квадрат 2×2 одного тира. Схлопывается;
  спецтайлов на поле НЕ рождает.
- **Бустер** — кнопка-инвентарь под полем (💣 Бомба / 🛸 Дрон / 🚀 Ракета / 🧲 Магнит) со
  счётчиком из сейва. Тап применяет (эффекты на поле — будущая фаза; сейчас тап → заглушка).
  Бустеры **на поле не появляются**.
- **Каскад** — повторный схлоп после досыпки в рамках одного хода.
- **Комбо** — число натуральных матч-групп за ход (и одновременных, и каскадных): «Комбо ×1»,
  «Комбо ×2», … — баннер над полем + накопленная сумма $. Бонус берётся по ФИНАЛЬНОМУ уровню
  (×1 +5%, ×2 +6%, …) ко всей сумме; деньги улетают в Баланс (карту) в конце хода.
- **Баланс** — главная валюта (карта «Total balance»). Растёт на ценность схлопнутых плиток.
- **💎 (алмазы)** — премиум-валюта (карта «Diamonds»).
- **Level** — уровень игрока (HUD; прокачка — будущее окно). **Energy** — энергия `N/100`
  (трата за ход + реген по таймеру — будущее; сейчас показ + статичный таймер). **Income ×N** —
  множитель дохода (= `investmentMultiplier`, старт ×1).
- **Офферы** — SALE (премиум-набор 💎) и Watch Ad (награда за рекламу) — заглушки.
- **Гравитация** — после схлопа уцелевшие плитки падают вниз, сверху досыпаются новые.

## Раскладка экрана (сверху вниз, дизайн-холст 390×844)

Шапка (аватар-хомяк + «Hamster Bank» + 🔔/⚙) → карта Баланс+Алмазы с маскотом → офферы
(SALE / Watch Ad) → строка Level / Energy / Income → поле 6×5 → ряд из 4 кнопок-бустеров →
нижнее меню 5 вкладок (Постройка / Задачи / **Игра** / Коллекции / Магазин; «Игра» активна).

## Текущая итерация

Перенесён макет «Hamster Bank»: синяя тема, шапка с маскотом, карта баланса, офферы, строка
Level/Energy/Income, поле 6×5, кнопки-бустеры, 5-вкладочное меню. Кор-петля match-3 (свайп,
схлоп 3+/2×2, каскады, комбо, полёт денег в карту, анти-дедлок) — рабочая. Заглушки (логика —
скоро): эффекты бустеров, энергия (трата/реген) + её таймер, прокачка уровня, офферы/реклама,
инвестиции, окна вкладок, 🔔. Прогрессия тиров — будущее.
