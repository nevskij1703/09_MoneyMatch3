# MoneyMatch3 — Game Design Document

f2p hybrid-casual **классический match-3** для РуСтор. Кор-механика — **свайп-обмен** двух
соседних денежных плиток: линии из 3+ по верт./гориз. и квадраты 2×2 авто-схлопываются в
Баланс, спецматчи (T/L → 💣, квадрат 2×2 → 🚀, линия-5 → 🧲) рождают бустеры-ОБЪЕКТЫ на поле;
сверху досыпаются новые плитки (гравитация, каскады). Сверху сидит Уоррен Баффет и реагирует на комбо.

## Глоссарий

- **Плитка / тир** — денежный объект на поле 6×6 (старт T1..T4; внутренняя стоимость `2^t`).
- **Свап** — обмен двух ортогонально соседних плиток свайпом; валиден, если создаёт матч
  (линию ≥ `minLine` или квадрат 2×2) или затрагивает бустер, иначе откатывается.
- **Матч** — линия ≥ `minLine` (=3) по верт./гориз. ИЛИ квадрат 2×2 одного тира.
- **Бустер** — САМОСТОЯТЕЛЬНЫЙ объект на поле (не накладка; `cells[i]=null`): 💣 **Бомба**
  (из T/L → взрыв 3×3), 🚀 **Ракета** (из 2×2 → сносит ряд/столбец, h/v рандом), 🧲 **Магнит**
  (из линии-5 → собирает весь тир). Срабатывает свайпом с любым соседом (матч не нужен; свайп
  двух бустеров активирует оба) или цепной реакцией (взрыв/ракета/магнит). Магнит не собирает
  бустеры: цель — тир соседа по свайпу либо случайная ближайшая плитка.
- **Каскад** — повторный схлоп после досыпки в рамках одного хода.
- **Комбо** — число натуральных матч-групп за ход (и одновременных, и каскадных; спецвзрывы НЕ
  считаются): «Комбо ×1», «Комбо ×2», … — баннер над полем + накопленная сумма $. Бонус берётся по
  ФИНАЛЬНОМУ уровню (×1 +5%, ×2 +6%, …) ко всей сумме; деньги улетают в Баланс в конце хода.
- **Баланс** — главная валюта. Растёт на ценность схлопнутых плиток.
- **💎 (алмазы)** — премиум-валюта (на будущие цены бустеров).
- **Гравитация** — после схлопа уцелевшие плитки падают вниз, сверху досыпаются новые.
- **investmentMultiplier** — множитель ценности сбора (будущие инвестиции его поднимают; старт 1).

## Текущая итерация

Реализована рабочая кор-петля классического match-3: свайп-обмен, схлоп линий 3+ и квадратов
2×2, бустеры-объекты (Бомба 3×3 из T/L, Ракета ряд/столбец из 2×2, Магнит — весь тир из линии-5)
с активацией свайпом и цепной реакцией, каскады с растущим комбо, анти-дедлок, чистый старт.
Заглушки: Баффет (статичная фигура + лёгкая реакция), 4 инвентарных
бустера, инвестиции, вкладки тулбара. Прогрессия тиров (новые элементы) — будущее.
